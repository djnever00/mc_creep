# Dae: Creep

Fabric `1.21.11` `dae_creep` mod.

## Current focus

Starting from vanilla creeper logic to manipulate:

- fuse timing
- explosion radius/fire behavior
- movement/aggro tuning
- rendering and spawn setup

## Build

Use your root wrapper:

```bash
./gradlew build
```

On Windows:

```powershell
.\gradlew.bat build
```
